# 麻雀点数計算ツール

M-League公式ルール準拠の麻雀点数計算練習Webアプリケーション。

**https://mahjong-calc-liard.vercel.app/**

## 機能

- **クイズ** — ランダム生成された手牌で点数計算を練習（初級・中級・上級）
- **検定** — 制限時間つきの実力テスト（1級〜初段）。全問正解で合格
- **自作問題** — 自分で手牌を作成・保存・出題
- **成績** — 不正解の自動カテゴリ分類、苦手分野ランキング、苦手克服モード
- **ルール** — 符計算・点数計算ルール、役一覧の早見表
- **クラウド同期** — X (Twitter) ログインで学習データをクラウドに保存
- **退会** — アカウント管理からクラウド・ローカルデータを全削除して退会

### 難易度

| レベル | 内容 |
|--------|------|
| 初級 | 基本役のみ（平和・断么九・一盃口など） |
| 中級 | 全役（三色・七対子・混一色・役満など） |
| 上級 | プロ試験対策（複雑な符計算を含む） |

※ 上級は最高位戦日本プロ麻雀協会のプロテスト過去問に基づいて作成しています。

### 検定レベル

| レベル | 出題範囲 | 制限時間 | 問題数 |
|--------|----------|----------|--------|
| 1級 | 基本役 | 30秒 | 5問 |
| 2級 | 全役 | 30秒 | 5問 |
| 3級 | プロ試験レベル | 30秒 | 5問 |
| 初段 | プロ試験レベル | 15秒 | 5問 |

## ルール準拠

このアプリは [M-League 公式ルール](https://m-league.jp/about/) の第3条・第4条に基づいた点数計算を実装しています。

### 主な特徴
- 場ゾロ込みの計算式: 基準点 = 符 × 2^(翻数 + 2)
- 切り上げ満貫（30符4翻、60符3翻など）
- 数え役満なし（11翻以上の通常役は三倍満が上限）
- 連風牌の雀頭は +2符（+4符ではない）
- 役満の倍数（ダブル役満・トリプル役満）対応
- 七対子は25符固定

> ※ 本アプリは M-League 公式とは関係ありません。ルール参照のため公開URLを引用していますが、独自に実装した非公式の練習用ツールです。

## データ保存について

- ローカルデータは **ブラウザの localStorage** に保存されます。
- X (Twitter) でログインすると、学習データが **Supabase にクラウド同期** されます。
- ログイン時はデバイス間でデータが共有されます。
- 退会するとクラウド・ローカルの全データが削除されます。
